
1. Install plugged

```bash
curl -fLo ~/.local/share/nvim/site/autoload/plug.vim --create-dirs \
https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

2. Install pynvim

```bash
python3 -m pip install --user --upgrade pynvim
```

3. Open ~/.config/nvim/init.vim

Run command:

```
:PlugInstall
```
